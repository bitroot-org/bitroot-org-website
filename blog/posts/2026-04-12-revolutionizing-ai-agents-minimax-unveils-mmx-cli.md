## Introduction to MMX-CLI
We just spotted an exciting update from MiniMax that's worth sharing with the community - the introduction of MMX-CLI, a innovative infrastructure designed specifically for AI agents. This new tool gives agents seven new senses, including image, video, voice, music, vision, search, and conversation, powered by MiniMax's full-modal stack.

## What Makes MMX-CLI Special
Here's what caught our attention about MMX-CLI: it's incredibly easy to use, with just two lines of code required to give an agent a voice. The features and benefits of MMX-CLI include:
* Enhanced agent capabilities with seven new senses
* Powered by MiniMax's full-modal stack
* Easy to use, with minimal code required
* Runs on existing Token Plans, with no additional costs

## Getting Started with MMX-CLI
To get started with MMX-CLI, developers can simply run two lines of code: npx skills add MiniMax-AI/cli -y -g and npm install -g mmx-cli. Then, they can tell their agent 'you have mmx commands available' and it will learn the rest.
