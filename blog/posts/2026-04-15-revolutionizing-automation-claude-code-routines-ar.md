## What Are Routines and Why Do They Matter?
Routines run on Claude's web infrastructure, so you don't have to keep your laptop open. This matters for developers because it means you can automate tasks without being tied to your local machine. What's more, routines can be customized with a prompt, a repository, and your connectors, making them highly adaptable to different use cases.

## Key Features and Benefits
Some key features of routines include:
* Configuring a routine once and running it multiple times
* Running routines on a schedule, from an API call, or in response to an event

* Routines run on Claude's web infrastructure, eliminating the need for a local machine
