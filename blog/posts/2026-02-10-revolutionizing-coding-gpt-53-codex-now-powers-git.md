## Introduction to GPT-5.3-Codex
We just spotted an exciting update from GitHub that's worth sharing with the community - GPT-5.3-Codex, the latest agentic coding model from OpenAI, is now rolling out in GitHub Copilot. This new model promises to deliver significant improvements in coding, agentic, and real-world capabilities.

## What's New and Notable
Here's what caught our attention about this update: GPT-5.3-Codex reaches new high scores on benchmarks, with improved reasoning and execution in complex workflows, resulting in up to 25% faster performance than its predecessor, GPT-5.2-Codex. Some key features and benefits include:
* Improved coding capabilities
* Enhanced agentic capabilities
* Faster performance in complex workflows

## Availability and Access
GPT-5.3-Codex will be available to Copilot Pro, Pro+, Business, and Enterprise users, and can be selected in the model picker in various GitHub platforms, including Visual Studio Code, github.com, and GitHub Mobile. To enable access, Copilot Enterprise and Copilot Business plan administrators must enable the GPT-5.3-Codex policy in Copilot settings.

## Implications and Potential Use Cases
The community is buzzing about the potential implications of this update, and we're excited to explore the possibilities. With GPT-5.3-Codex, developers can expect to work more efficiently, focusing on high-level tasks while the AI model handles the heavy lifting. This could lead to significant advancements in areas like automated coding, code review, and project management.
